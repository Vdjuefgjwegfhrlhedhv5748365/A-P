3) Изменения в программе:
    1. Функция print_book, изначально реализованная как отдельная, была интегрирована в функцию print_catalog.
    2. Функция freeCatalog удалена. Вместо создания отдельной функции для освобождения памяти, вызов free(catalog) был осуществлен непосредственно в точке использования.
    3. Подключение стандартных библиотек (stdio.h, stdlib.h, string.h) было перенесено из заголовочного файла function.h в соответствующие .c файлы реализации.
    4. Был удалён лишний вызов freeCatalog(catalog) на строке 177 (ранее строка 166).
    5. Убран коктейль из scanf и fgets: теперьв программе используется только fgets.
